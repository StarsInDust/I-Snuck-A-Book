### "Web server not running" Error:

1. Make sure you ran `start_web_server.bat`
2. Check that Python is installed on your system
3. Verify no antivirus is blocking the server
4. Try running `python web_server.py` manually

### Server Won't Start:

1. Install required Python packages: `pip install flask pymupdf pillow`
2. Check if port 8000 is already in use
3. Run as administrator if needed

### Compression Not Working:

1. Verify server is running (green status in interface)
2. Try with a different PDF file
3. Check file size limits (server handles most reasonable PDFs)
